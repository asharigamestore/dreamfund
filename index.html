FAQ

Pertanyaan yang Sering
Ditanyakan

Ya, DreamFund 100% gratis selamanya. Tidak ada biaya berlangganan, tidak ada iklan yang mengganggu, dan tidak ada fitur premium yang terkunci. Kami percaya semua orang berhak mengatur keuangan mereka dengan baik.

Sangat aman. Semua data tersimpan di LocalStorage browser milikmu sendiri, bukan di server kami. Data tidak pernah dikirim ke internet, sehingga privasimu sepenuhnya terjaga. Kamu juga bisa export data kapan saja.

Karena data tersimpan di LocalStorage per browser, untuk mengakses dari perangkat lain kamu bisa menggunakan fitur Export/Import JSON di menu Pengaturan. Export data dari perangkat lama, lalu import di perangkat baru.

Bukan. DreamFund adalah aplikasi tracking dan planning keuangan. Kami tidak menyimpan uang, tidak melakukan transaksi, dan tidak terhubung dengan bank manapun. Kami hanya membantumu mengatur dan memantau target keuangan secara visual.

Kamu bisa mereset semua data melalui menu Pengaturan > Reset Data. Pastikan untuk export data terlebih dahulu jika ingin menyimpan backup. Reset akan menghapus semua target, wishlist, dan riwayat tabungan.
